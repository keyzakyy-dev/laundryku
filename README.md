## 📋 Deskripsi

**Sistem Pengelolaan Laundry** adalah aplikasi web sederhana namun powerful yang dibangun menggunakan **PHP Native** dan **MySQL**. Dirancang khusus untuk membantu pemilik usaha laundry dalam mengelola data pelanggan, layanan, pesanan, dan status pengerjaan secara efisien.

Dengan tampilan dashboard yang bersih dan modern, sistem ini memungkinkan pencatatan transaksi laundry menjadi lebih **rapi, cepat, dan mudah digunakan** — bahkan untuk pengguna yang tidak memiliki latar belakang teknis.

---

## ✨ Fitur

| Fitur | Deskripsi | Status |
|-------|-----------|--------|
| 👥 **Manajemen Pelanggan** | Tambah, edit, hapus, dan kelola data pelanggan dengan mudah | ✅ Aktif |
| 🧼 **Daftar Layanan** | Tampilkan berbagai paket layanan laundry yang tersedia | ✅ Aktif |
| 📦 **Manajemen Pesanan** | Kelola data pesanan laundry dari masuk hingga selesai | ✅ Aktif |
| 🔄 **Tracking Status** | Ubah dan pantau status pesanan secara real-time | ✅ Aktif |
| 🗑️ **Hapus Data** | Hapus data pelanggan atau pesanan dengan konfirmasi aman | ✅ Aktif |
| 📊 **Riwayat Terbaru** | Lihat ringkasan pesanan terbaru di dashboard utama | ✅ Aktif |
| ⭐ **Layanan Populer** | Analisis layanan yang paling sering dipilih pelanggan | ✅ Aktif |
| 🎨 **UI Modern** | Tampilan clean dashboard dengan CSS custom yang responsif | ✅ Aktif |

## 🛠️ Teknologi Stack

| Kategori | Teknologi |
|----------|-----------|
| **Backend** | PHP Native |
| **Database** | MySQL |
| **Frontend** | HTML5, CSS3, JavaScript |
| **UI Library** | SweetAlert2 (popup interaktif) |
| **Server Lokal** | XAMPP / Laragon |
